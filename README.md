# `dburl`

Easily connect to a database url like `postgres://username:password@hostname:1234/dbname`

#### Currently Supported
* Postgres

...but adding a new one is easy!

## Installation

Install the gem with `gem install dburl`.

## Usage

`$ dburl <your database url>` simple!
